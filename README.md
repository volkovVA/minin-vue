# vue-crm
## Creating a CRM application on VUE and firebase

* Vue
* Vue CLI
* Vue Router
* Vuex
* Firebase
* Vue-chartjs
* Materialize

```
npm install
```


```
npm run build
```
### Lints and fixes files

```
npm run lint
```
